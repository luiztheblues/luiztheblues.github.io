› otto-pkg
- Um sistema estilo BSD Ports para o Copacabana OpenLinux, que visa ser automatizado à moda Portage.
Começou como uma tentativa de se fazer algo completamente independente, mas como a base antiga de código estava ficando muito bagunçada, decidi engavetar a ideia anterior e simplesmente fazer um "auto-fork" (isso existe?) do uberports.
Futuramente, eu tentarei fazer minha ideia original em uma linguagem um pouco mais competente (como Go ou C) e criar algum script que converta os arquivos e pacotes atuais para o novo formato, afim de não quebrar completamente a portabilidade com o otto-pkg atual.
